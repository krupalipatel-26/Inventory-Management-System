# 📦 Inventory Management System

A Java console-based Inventory Management System developed using Core Java and HashMap.

## Features

- User Login Authentication
- Add Products
- View Products
- Update Product Details
- Delete Products
- Sell Products
- Automatic Bill Calculation
- Low Stock Warning
- Console Menu Driven Interface

---

## Technologies Used

- Java
- HashMap
- Scanner
- OOP Concepts
